# Elementa

A beautiful physics sandbox game for mobile browsers. Place materials and watch them interact with realistic physics.

## Materials

**Solids:** Sand, Stone, Wood
**Liquids:** Water, Oil, Lava
**Gas/Energy:** Fire, Steam, Smoke
**Special:** Plant

## Features

- Realistic physics with gravity and density
- Material interactions (fire burns wood, water extinguishes fire, lava + water = stone)
- Beautiful pixel art aesthetic
- Mobile-friendly touch controls
- Pinch to adjust brush size

## Controls

- **Tap/Drag:** Place particles
- **Pinch:** Adjust brush size
- **Toolbar:** Select materials by category
- **Eraser:** Remove particles
- **Clear:** Reset canvas
